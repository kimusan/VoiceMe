# VoiceMe - easy, on-device, privacy focused voice dictation for Android

VoiceMe is an easy to use Android app that makes it effortless to do voice dictation anywhere on an A½ndroid device where a keyboard would normally be used.
A floating button appear whenever the keyboard apears. Holding it down (can be configured to be a toggle) will start the microphone listening for your voice.
The voice is quickly converted into text and written to the currently selected input field. The convertion is happening 100% on device and uses the Parakeet
family of LLM Automatic Voice Recognition (ASR) models (other OpenAI whisper compatiple models could be in the list too).

## Features

- Easy onboarding that lets the user download model to use
- Hidden away when not in use, but shown as soon as the keyboard is around
- Configurable size/color of the button
- Button is dragable if it needs to be places elsewhere. 
- Can be bound to a hardware button on the device if one is available to configure (Direct activity link to start dictation)
- near realtime transcription of what you say and written directly to the active 
- privacy focussed so no external model usage or telemetry sent out of the device
